# ticktracker

This program is used to calculate trends in Robintrack data, available online. It takes .csv files in the format that they are outputted through the Robintrack data, creates folders with respect to market cap and the classification of industry they work in(courtesy of Bloomberg data) and creates a simple percentage change per day based on the Robintrack data.
